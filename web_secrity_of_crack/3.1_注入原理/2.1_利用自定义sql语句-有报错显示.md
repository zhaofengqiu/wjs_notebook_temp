### 基于内容的条件语句
通过if语句实现修改参数，从而实现基于页面进行条件语句
```sql
id = 12+(case when(system_user='sa') then 1 else 0 end)
```
如果查询到的用户是12，那么system_user就不是’sa‘;如果查询到的用户是13,那么最后system_user就是'sa'.
